# Data Analysis Projects

This repository contains multiple Data Analysis and Machine Learning projects developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

## Projects

### 1. IPL Data Analysis

* Exploratory Data Analysis (EDA) on IPL datasets.
* Analysis of match winners, toss decisions, venues, player performances, top scorers, and bowling figures.
* Data visualization using Matplotlib and Seaborn.

### 2. COVID Hospitalization Prediction

* Machine Learning project using Logistic Regression.
* Predicts whether a patient may require hospitalization based on medical indicators.
* Includes data preprocessing, feature engineering, model training, and evaluation.
